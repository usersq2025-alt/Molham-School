# حملة تجهيز مدرسة ملهم للأيتام — موقع الداشبورد

موقع صفحة واحدة (index.html) جاهز للنشر على GitHub Pages، يعرض قصة الحملة والمؤشرات وخطة الصرف.

## 1) هيكل الملفات

```
├── index.html                 ← الصفحة الكاملة (لا تحتاج لمسها للنشر)
└── assets/
    ├── images/                ← ضع كل الصور هنا بالأسماء المحددة أدناه
    └── video/                 ← اختياري، راجع قسم الفيديو أدناه
```

## 2) أسماء الصور المطلوبة بالضبط

ضع كل صورة داخل مجلد `assets/images/` **بنفس الاسم والامتداد** التالي (jpg أو png، الأحرف صغيرة تمامًا):

| اسم الملف | أين تظهر | المقاس المفضل |
|---|---|---|
| `logo.png` | شعار الفريق في الهيدر | مربع، خلفية شفافة، ~400×400 |
| `hero.jpg` | الصورة الرئيسية أعلى الصفحة | أفقية عريضة، 1600×1200 تقريبًا |
| `story-1.jpg` | صورة بورتريه الطالب في قصة القسم الأول | مربعة، 800×800 |
| `story-2.jpg` | صورة صغيرة في معرض القسم الأول | مربعة |
| `story-3.jpg` | صورة صغيرة في معرض القسم الأول | مربعة |
| `activity-1.jpg` | صورة نشاط/فريق كرة القدم | مربعة |
| `building-current-1.jpg` | المبنى الحالي (واجهة/فناء) | أفقية 16:10 |
| `building-current-2.jpg` | المبنى الحالي (داخل الصفوف) | أفقية 16:10 |
| `gallery-1.jpg` `gallery-2.jpg` `gallery-3.jpg` | معرض توثيقي في قسم الأثر | مربعة |

> إن لم تتوفر صورة بعد، **لا تحذف السطر من الكود** — الصفحة تعرض تلقائيًا مربعًا رماديًا متقطعًا مكتوبًا عليه اسم الملف المطلوب، بدل أن تظهر صورة مكسورة. بمجرد رفع الصورة بنفس الاسم، تظهر مكانه فورًا بدون أي تعديل آخر على الكود.

**تحتاج صورًا إضافية غير مذكورة أعلاه؟** أضف وسم `<img>` جديدًا بنفس نمط `photo-slot` الموجود في الكود، بأي اسم ملف تريده.

## 3) الفيديو

لا يُنصح برفع ملفات فيديو مباشرة داخل مستودع GitHub (حجم كبير + بطء الاستضافة). بدلًا من ذلك:

1. ارفع الفيديو على يوتيوب (كفيديو غير مدرج/Unlisted إن أردت خصوصية) أو استخدم رابط مشاركة من Google Drive.
2. افتح `index.html`، ابحث عن السطر:
   ```js
   videoUrl: "",
   ```
   وضع رابط التضمين (embed) بين علامتي التنصيص، مثال يوتيوب:
   ```js
   videoUrl: "https://www.youtube.com/embed/XXXXXXXXXXX",
   ```
3. احفظ الملف — الفيديو سيظهر تلقائيًا في قسم "الأثر المتوقع".

## 4) تعديل الأرقام والروابط بدون لمس التصميم

كل الإعدادات القابلة للتعديل مجمّعة في أعلى الكود داخل `index.html`، ابحث عن:

```js
const CONFIG = { ... }
```

من هناك تقدر تعدّل: المبلغ المُجمَّع حتى الآن (`fundingRaised`)، رابط التبرع (`donateUrl`)، رابط ألبوم الصور والتوثيقات على Google Drive (`driveAlbumUrl`)، ورابط الفيديو (`videoUrl`)، وبنود توزيع مبلغ 300,000$ (`budgetItems`) — هذه الأخيرة تقديرية حاليًا، بانتظار عروض أسعار فعلية للترميم والتأثيث.

## 5) النشر على GitHub Pages (خطوة بخطوة)

1. أنشئ مستودعًا جديدًا (Repository) على GitHub، مثلًا باسم `mulham-orphans-school`.
2. ارفع محتويات هذا المجلد بالكامل (`index.html` ومجلد `assets`) إلى المستودع — إما عبر السحب والإفلات في واجهة GitHub (زر Add file → Upload files)، أو عبر Git.
3. من إعدادات المستودع: **Settings → Pages**.
4. تحت "Build and deployment"، اختر Source: **Deploy from a branch**، والفرع **main** والمجلد **/ (root)**.
5. احفظ، وانتظر دقيقة إلى دقيقتين. سيظهر رابط الموقع أعلى نفس الصفحة بالشكل:
   `https://<اسم-المستخدم>.github.io/<اسم-المستودع>/`

أي تعديل لاحق (رقم جديد، صورة جديدة) يكفي رفعه لنفس المستودع، والموقع يتحدث تلقائيًا خلال دقائق.
